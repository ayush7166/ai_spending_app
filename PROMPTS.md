# Prompt Documentation

## Overview

This document contains prompt design ideas and example AI interaction templates for future integration of large language models into the AI Spend Audit Engine.

The current system uses rule-based recommendation logic. Future versions may use AI-generated responses for conversational subscription analysis and recommendation summaries.



## System Prompt

You are an AI subscription optimization assistant.

Analyze the user's current AI subscription plan and recommend lower-cost alternatives with equivalent capability levels across providers.

Provide clear, concise, and professional recommendations focused on cost optimization and usability.



### Example 

I currently use ChatGPT Team for 5 users.
Suggest a cheaper alternative with similar capabilities.
